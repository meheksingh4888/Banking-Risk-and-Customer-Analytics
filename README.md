# 🏦 Banking Risk Customer Analytics

An end-to-end Data Analytics project built using PostgreSQL, SQL, Python, and Power BI to analyze banking operations, customer behavior, loan portfolios, transaction patterns, and financial risk.

This project demonstrates the complete data analytics lifecycle—from database design and data generation to SQL analysis, Python ETL, interactive Power BI dashboards, and business reporting.

---

# Project Overview

The Banking Risk Customer Analytics project helps financial institutions gain valuable insights into:

- Customer Segmentation
- Banking Transactions
- Loan Portfolio Analysis
- Credit Risk Assessment
- Branch Performance
- Merchant Analytics
- Fraud Detection
- Executive Business Reporting

The project simulates a real-world banking environment using synthetic datasets generated with Python.

---

# Objectives

- Design a normalized banking database
- Generate realistic banking data
- Perform SQL-based business analysis
- Build an ETL pipeline using Python
- Create interactive Power BI dashboards
- Identify financial risks and business opportunities
- Generate executive-level reports

# How to Run

## Clone Repository

```bash
git clone https://github.com/yourusername/Banking-Risk-Customer-Analytics.git
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Configure PostgreSQL

Update

```
python/config/database.py
```

with your database credentials.

---

## Run Python Analysis

```bash
python -m python.Analysis.customer_analysis

python -m python.Analysis.account_analysis

python -m python.Analysis.transaction_analysis

python -m python.Analysis.loan_analysis

python -m python.Analysis.branch_analysis

python -m python.Analysis.merchant_analysis

python -m python.Analysis.risk_analysis

python -m python.Analysis.executive_summary
```
